安装和配置

安装配置步骤：
  • 下载和安装JDK（6.0或以上)
  • 设置环境变量JAVA_HOME为JDK所在目录
  • 将JAVA_HOME/bin添加到PATH环境变量中
  • 设置环境变量SOYA_HOME为soya所在目录
  • 将SOYA_HOME/bin添加到PATH环境变量中
您可以尝试在命令行中输入以下命令来测试配置是否正确
    soya -version
键入以下命令以运行一个soya脚本
    soya aScript.soya

在IDE中运行Soya

通过带有main方法的类Main来启动运行Soya脚本，具体可根据下列设置方式来运行Soya脚本
    java org.soya.tools.Main [Source File] [arguments]

在命令行中运行Soya

根据您所使用的操作系统可以选择'soya'或'soya.bat'来运行脚本，只要您正确安装配置了Soya，可以通过下列方式运行Soya
    soya [Source File] [arguments]
